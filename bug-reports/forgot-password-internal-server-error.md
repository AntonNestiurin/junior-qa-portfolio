# Bug Report

## Title
Internal server error occurs when submitting invalid input

## Bug Type
Functional / Validation

## Severity
High

## Priority
High

## Environment
- URL: https://the-internet.herokuapp.com/
- Browser: Chrome
- OS: Windows 11
- Viewport size: any

## Preconditions
- User is on the site "The Internet"

## Steps to Reproduce
1. Navigate to the Forgot Password page  
2. Type “1” in the email field  
3. Press “Retrieve password”

## Actual Result
The user is navigated to a page displaying: “Internal Server Error”

## Expected Result
The user should remain on the Forgot Password page, and the system should display an “Invalid email address” message
