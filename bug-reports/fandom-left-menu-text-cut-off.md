# Bug Report

## Title
Text under the compass icon in the left navigation menu is cut off

## Bug Type
UI / UX

## Severity
Medium

## Priority
Medium

## Environment
- URL: https://anime-characters-fight.fandom.com/
- Browser: Chrome
- OS: Windows 11
- Viewport size: 400 Ã— 642

## Preconditions
- User is on an ACF Wiki page on the Fandom website

## Steps to Reproduce
1. Open the Fandom ACF Wiki page
2. Locate the left navigation menu
3. Find the compass icon in the menu

## Actual Result
The text under the compass icon is partially cut off and cannot be fully read.  
There is no option to resize or expand the menu.

## Expected Result
The text under the compass icon should be fully visible.  
If the text does not fit, it should wrap to the next line or the menu width should be adjustable.
