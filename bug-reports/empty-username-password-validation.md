# Bug Report

## Title
Incorrect validation message after logging in with empty Username and Password fields

## Bug Type
Validation

## Severity
Medium

## Priority
Medium

## Environment
- URL: https://the-internet.herokuapp.com/
- Browser: Chrome
- OS: Windows 11
- Viewport size: any

## Preconditions
- User is on the site "The Internet"

## Steps to Reproduce
1. Navigate to the Form Authentication page  
2. Leave Username and Password fields empty  
3. Click the Login button

## Actual Result
A message shows: “Your username is invalid.”

## Expected Result
A message shows: “Please fill in all required fields.”
