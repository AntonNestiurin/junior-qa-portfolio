# Bug Report

## Title
Advertisement at the top of the main page is not displayed correctly

## Bug Type
Mobile UI

## Severity
Low

## Priority
Low

## Environment
- URL: https://edition.cnn.com/
- Browser: Chrome
- OS: Samsung Galaxy A23
- Viewport size: any

## Preconditions
- User is on the main page of the CNN site

## Steps to Reproduce
1. Open the main page of CNN  
2. Observe the advertisement banner at the top of the page

## Actual Result
A grey rectangle is displayed at the top of the main page instead of an advertisement

## Expected Result
An advertisement should be displayed correctly
