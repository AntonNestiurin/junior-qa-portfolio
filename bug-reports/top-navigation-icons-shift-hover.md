# Bug Report

## Title
Icons in the top navigation menu shift slightly on hover

## Bug Type
UI / UX

## Severity
Low

## Priority
Low

## Environment
- URL: https://www.amazon.com/
- Browser: Chrome
- OS: Windows 11
- Viewport size: any

## Preconditions
- User is on the main page of the site

## Steps to Reproduce
1. Open the Amazon main page
2. Hover the mouse cursor over any icon in the top navigation menu

## Actual Result
Icons in the top navigation menu move slightly when hovering over them

## Expected Result
Icons in the top navigation menu should remain stable and not change their size or position on hover
